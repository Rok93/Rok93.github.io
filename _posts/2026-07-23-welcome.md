코드로 검증하며 배우는 과정을 기록하려고 블로그를 열었습니다.

이 블로그는 두 가지 스킨을 가지고 있습니다. 오른쪽 위 **테마** 버튼으로 전환해보세요.

- **Hyde** — 컬리 퍼플 사이드바의 2단 구성 (기본)
- **Custom** — 잉크&스틸 에디토리얼 단일 컬럼 (계속 다듬는 중)

```java
// 앞으로 다룰 주제의 예: 이벤트의 트랜잭션 경계
@EventListener
public void onSomethingHappened(SomethingEvent e) {
    // 커밋 전, 같은 트랜잭션 — 원자적으로 처리
}
```

> load-bearing 여부는 상상하지 말고 "누가 그 값을 읽고 무슨 결정을 하는지"를 코드에서 확인해 정하라.
